# stm32-fingerprint-rfid-doorlock
An STM32-based smart door lock system using FreeRTOS, fingerprint authentication(AS608), RFID cards(RC522), keypad password entry, ultrasonic(HC-SR04) auto-open detection, servo motor control, and LCD16x2 i2c module display. The project also supports local flash storage for enrolled fingerprint IDs and registered RFID cards.
